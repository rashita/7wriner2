# 7wriner

7本のLineを横に並べて管理するアウトライナー型メモツールです。  
単一のHTMLファイルで動作し、サーバー不要。データはブラウザのlocalStorageに保存されます。

---

https://rashita.github.io/7wriner2/

## コンセプト

7wrinerは**7本のLine**を持ちます。画面には常に中央1本＋左右各1本の計3本が表示され、`⌘+←/→` でLineをカルーセル状にシフトできます。

7本というハードリミットが設計の核心です。無限にLineを増やすのではなく、「どのLineに何を置くか」を判断すること自体が思考の整理になります。溢れたLineは **Send/Call** 機能で一時退避できます。

---

## 基本的な使い方

### Lineの操作
- 画面上部のドット（●）で現在の位置を確認できます
- `⌘+←/→` でLineを左右にシフト（循環します）
- LineのタイトルをクリックするとそのLineがCenter（中央）に移動します

### アイテムの入力
- 画面上部の入力欄に文字を入力し、`⌘+Enter` で中央のLineに追加されます
- `Enter` は通常の改行として使えるので、複数行のテキストも入力できます

### アイテムの編集
- アイテムのテキスト部分はその場で直接編集できます
- **ダブルクリック**で全画面エディタが開き、原稿を書くように集中して編集できます（Escまたはオーバーレイクリックで保存・終了）

### アイテムの完了
- アイテム左の `◦` をクリックすると完了状態（取り消し線）になります

## Send / Call（一時退避）

7本のLineが埋まってきたとき、使用頻度の低いLineを一時的に退避できます。

- **Send** — 中央LineのアイテムをすべてTemporary Placementに退避し、Lineを空にします
- **Call** — 退避済みのLineを一覧から選んで呼び出せます
  - **呼び出す** — 選択したLineのアイテムを中央Lineに追加し、退避領域から削除します
  - **テンプレート利用** — Lineのタイトルも引き継いで追加します（退避領域には残ります）
  - **削除** — 退避領域から削除します

---

## 複数選択

`⌘+クリック` でアイテムを複数選択できます。複数のLine跨ぎも可能です。  
選択すると画面下部にアクションバーが表示されます。

| アクション | 説明 |
|---|---|
| Merge | 選択したアイテムを1つに結合（同一Line内、上から順に改行で連結） |
| ← Move / Move → | 選択アイテムを隣のLineに一括移動 |
| Delete | 選択アイテムを一括削除 |
| ✕ | 選択解除 |

---

## メニュー

右上の `menu` ボタンから以下の操作が行えます。

| 項目 | 説明 |
|---|---|
| Export | 全Lineの内容をテキストとして書き出し（コピー可） |
| Backup | JSONファイルとしてダウンロード |
| Import | バックアップJSONを読み込んで復元 |
| Clear All | 全データを削除 |

## ショートカットキー

### Line操作

| キー | 動作 |
|---|---|
| `⌘ + ←` | Lineを左にシフト |
| `⌘ + →` | Lineを右にシフト |

### アイテム入力・編集

| キー | 動作 |
|---|---|
| `⌘ + Enter` | 上部入力欄から中央Lineにアイテムを追加 |
| `Enter`（入力欄内） | 改行 |
| `Enter`（アイテム内） | 直下に新しいアイテムを追加 |
| `Backspace`（空アイテム） | そのアイテムを削除 |
| `Esc`（全画面エディタ） | 保存して閉じる |

### アイテム移動

| キー | 動作 |
|---|---|
| `⌘ + Shift + ←` | 選択アイテムを左のLineに移動 |
| `⌘ + Shift + →` | 選択アイテムを右のLineに移動 |
| `⌘ + Shift + ↑` | 選択アイテムをLine内で一つ上に移動 |
| `⌘ + Shift + ↓` | 選択アイテムをLine内で一つ下に移動 |

### 複数選択中

| キー | 動作 |
|---|---|
| `⌘ + クリック` | アイテムを複数選択・解除 |
| `M` | 選択アイテムをMerge |
| `[` | 選択アイテムを左のLineに移動 |
| `]` | 選択アイテムを右のLineに移動 |
| `Backspace` / `Delete` | 選択アイテムを削除（テキスト編集中を除く） |
| `Esc` | 選択解除 |

---

## データについて

- データはブラウザの `localStorage` に自動保存されます
- 異なるブラウザ・デバイス間での同期には対応していません
- 定期的に **Backup** からJSONファイルをダウンロードしておくことをおすすめします

---

## 動作環境

単一のHTMLファイルです。モダンブラウザ（Chrome / Firefox / Safari）で動作します。  
サーバーは不要で、ファイルをダブルクリックしてブラウザで開くだけで使えます。
